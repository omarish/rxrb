# Rxrb

Port of: https://github.com/rjbs/Rx/tree/master/ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rxrb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rxrb
